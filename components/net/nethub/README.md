# Resource Usage

## nethub module

- code size:  < 4   KBytes
- ram usage:  < 0.4 KBytes
- buffer: configurable, default configuration <24 K (dnld buffer zero copy)

## sdio driver

- code size:  < 9 KBytes
- ram usage:  < 5 KBytes
- buffer: configurable, default configuration <50 K (upld buffer zero copy)
